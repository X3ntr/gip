Opzetten van Pelican

Pelican maakt gebruik van het Python framework dat we kunnen installeren met behulp van Chocolatey.
$choco install python

Nadat we Python aan onze path variabele hebben toegevoegd kunnen we met de Python package manager pip Pelican en zijn bijhorende plugins en dependenties installeren.
$pip install pelican
$pip install jinja2
$pip install markdown
$pip install feedgenerator
$pip install pygments
$pip install docutils
$pip install pytz
$pip install blinker
$pip install unidecode
$pip install six
$pip install markupsafe
$pip install python-dateutil

Nu Pelican is geïnstalleerd kunnen we een "Environment" aanmaken.
$pelican-quickstart
Pelican zal vragen om 6 vragen te beantwoorden, namelijk: Waar je je nieuwe website wil maken, de titel van de website, de auteur van de website, de standaard taal van de website (default is Engels), URL prefix, pagination.

In de aangemaakte directory vinden we onze settings terug (pelicanconf.py), een content en een output directory en een aantal andere gegenereerde files. In de content directory maken we een nieuwe directory "pages".

Toevoegen van content en stylen van de website

Posts en pages worden opgemaakt met Markdown en daarna in de "content" of "pages" directory opgeslagen als een .md file.
Om onze website te stylen/opmaken kunnen we bestaande "themes" importeren. Hier zijn er een aantal. Om deze themes te gebruiken moeten ze lokaal worden opgeslagen.
In dit geval kunnen we Git gebruiken om de repository te clonen. Nu de themes lokaal beschikbaar zijn kunnen we ze bewerken of gebruiken.
$pelican-themes -vi {path naar theme} om te installeren
$pelican-themes -l dit geeft ons een lijst van beschikbare themes
$pelican -t {theme} om te gebruiken

Om onze site te genereren gebruiken we het volgende commando
$pelican content

Om onze site te bekijken vanuit de webbrowser:
$cd ~\yoursite\output
$python -m http.server

Open je webbrowser en surf naar http://localhost:8000
